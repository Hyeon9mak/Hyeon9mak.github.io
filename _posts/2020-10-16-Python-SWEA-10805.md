### 초기 접근
```python
T = int(input())
for test_case in range(1, T+1):
    N, Q = map(int, input().split())
    ball = {i:0 for i in range(1, N+1)}
    ball[1], ball[2] = 2, 1

    for q in range(Q):
        s, d = map(int, input().split())
        ball[s], ball[d] = ball[d], ball[s] #swap
        
        if ball[s] == 2:
            if s >= 2: ball[s-1] = 1
            if s < N: ball[s+1] = 1
        if ball[d] == 2:
            if d >= 2: ball[d-1] = 1
            if d < N: ball[d+1] = 1
        print(ball)
    print('#'+str(test_case), sum(v for k,v in ball.items())-1)
```
