## 풀이
```python
def solution(numbers):
    from itertools import combinations
    return sorted(list(set(map(lambda x: x[0]+x[1],list(combinations(numbers, 2))))))
```
**파이써닉**하게 짜는데 집중한 코드.  
itertools를 사용 못하게 하는 코테도 있다는데,  
이리도 간편하고 재밌는 걸?
