## 풀이
```python
def solution(scoville, K):
    import heapq
    answer = 0
    while scoville[0] < K :
        s1, s2 = heapq.heappop(scoville), heapq.heappop(scoville)
        heapq.heappush(scoville, s1+s2*2)
        answer +=1
    
    return answer
```
