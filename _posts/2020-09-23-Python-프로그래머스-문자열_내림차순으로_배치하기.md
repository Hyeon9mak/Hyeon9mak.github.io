## 문제

## 풀이
```python
def solution(s):
    return "".join(sorted(s, reverse=True))
```
문제 조건 중에 "대문자는 더 작은 것으로 간주합니다."가 있길레  
ASCII를 기준으로 정렬되겠거니 하고 reverse=True 옵션으로 정렬.  
"대문자는 더 큰 것으로 간주합니다." 였으면 다른 짓을 했어야겠지?
