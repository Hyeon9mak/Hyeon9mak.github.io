## 문제
[https://www.acmicpc.net/problem/10773](https://www.acmicpc.net/problem/10773)

## 풀이
기본 스택 구현.  

- 0을 입력 받으면 pop
- 그 외엔 push
- 마지막에 스택의 합을 출력

```python
stack = []
lines = int(input())

for i in range(lines):
    number = int(input())
    if number == 0: stack.pop()
    else: stack.append(number)

print(sum(stack))
```
